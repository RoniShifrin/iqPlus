IQ PLUS — Quick Run
====================

1. START BACKEND
   cd BACK
   venv\Scripts\activate
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

2. START FRONTEND  (new terminal)
   cd FRONT
   npm install
   npm run dev

3. SEED DEMO DATA  (new terminal, backend must be running)
   cd BACK
   venv\Scripts\python scripts/seed_large_demo.py

4. OPEN APP
   http://localhost:5173
   API docs: http://localhost:8000/docs

5. LOGIN  (email + password)
   Admin   : admin.demo@large.iqplus.dev   / Admin123!
   Teacher : teacher.math@large.iqplus.dev / Teacher123!
   Student : student.star@large.iqplus.dev / Student123!
   Parent  : parent.one@large.iqplus.dev   / Parent123!

6. CLEAR DEMO DATA
   venv\Scripts\python scripts/clear_large_demo.py

7. RESEED (clear + regenerate)
   venv\Scripts\python scripts/reseed_demo.py large

------------------------------------------------------------
KNOWN ISSUES & FIXES
------------------------------------------------------------

A. bcrypt incompatible with passlib
   Symptom : "password cannot be longer than 72 bytes" error
   Fix     : venv\Scripts\python -m pip install "bcrypt<4.0.0"

B. Frontend "Failed to fetch" on login
   Symptom : Login form shows "Failed to fetch"
   Fix     : Check FRONT\.env.local — must be:
             VITE_API_URL=http://127.0.0.1:8000
             Then restart: Ctrl+C  →  npm run dev

C. Seed fails with DuplicateKeyError (partial previous run)
   Symptom : "E11000 duplicate key error" on seed
   Fix     : Run clear first, then seed:
             venv\Scripts\python scripts/clear_large_demo.py
             venv\Scripts\python scripts/seed_large_demo.py
             (if clear finds nothing, see RUNNING_GUIDE.md troubleshooting)

D. MongoDB SSL error (TLSV1_ALERT_INTERNAL_ERROR)
   Symptom : SSL handshake failed on startup
   Cause   : Network/ISP intercepting TLS on port 27017
   Fix     : Switch to mobile hotspot or disable antivirus SSL scanning
